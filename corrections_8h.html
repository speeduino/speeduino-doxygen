◆ correctionAccel()

uint16_t correctionAccel ( )

Acceleration enrichment correction calculation.

Calculates the % change of the throttle over time (%/second) and performs a lookup based on this When the enrichment is turned on, it runs at that amount for a fixed period of time (taeTime)

Returns
uint16_t The Acceleration enrichment modifier as a %. 100% = No modification. As the maximum enrichment amount is +255%, the overall return value from this function can be 100+255=355. Hence this function returns a uint16_t rather than byte

◆ correctionASE()

byte correctionASE ( )

Afer Start Enrichment calculation.

This is a short period (Usually <20 seconds) immediately after the engine first fires (But not when cranking) where an additional amount of fuel is added (Over and above the WUE amount)

Returns
uint8_t The After Start Enrichment modifier as a %. 100% = No modification.
