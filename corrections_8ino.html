◆ correctionAccel()

static int16_t correctionAccel ( )
inlinestatic

Acceleration enrichment correction calculation.

Calculates the % change of the throttle over time (%/second) and performs a lookup based on this When the enrichment is turned on, it runs at that amount for a fixed period of time (taeTime)

Returns
int16_t The Acceleration enrichment modifier as a %. 100% = No modification. As the maximum enrichment amount is +255%, the overall return value from this function can be 100+255=355. Hence this function returns a int16_t rather than byte
